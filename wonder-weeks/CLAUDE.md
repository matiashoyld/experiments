# Wonder Weeks Baby Tracker

Baby development leap tracker inspired by the Wonder Weeks concept. Shows a personalized timeline of the 10 developmental leaps based on the baby's due date.

## Tech stack

- Vanilla HTML/CSS/JS (single `index.html`)
- No dependencies

## How to run

Open `index.html` in a browser, or:

```bash
cd wonder-weeks/
npx serve .
```

## Features

- Due date input to personalize the timeline
- Color-coded leap chart (stormy/sunny periods)
- Leap detail cards with descriptions and skills
- Mobile-first responsive design
- Data persisted in localStorage
